# StageVault Dark Theme — Quick Mapping

Краткая карта тёмной темы **Warm Nocturne**: какой цвет на какой элемент ставить.

---

## Основные цвета

- **Background** — `#171411`
- **Soft background** — `#1d1916`
- **Surface / card** — `#221d19`
- **Surface hover / active** — `#2b241f`
- **Strong surface** — `#342b24`
- **Border / divider** — `#42372f`

- **Primary text** — `#f1e6d9`
- **Secondary text** — `#cdbbaa`
- **Muted text / placeholder** — `#a18f7f`

- **Primary accent** — `#b48d72`
- **Soft accent** — `#d8b69a`
- **Olive accent** — `#86906a`

- **Done / completed** — `#9f8474`
- **Danger / destructive** — `#c76868`

---

## Что куда ставить

### 1. Общий фон приложения
- фон страницы — `#171411`
- внутренние большие секции — `#1d1916`

### 2. Navbar / header
- фон — `#221d19`
- нижняя граница — `#42372f`
- логотип / главный текст — `#f1e6d9`
- неактивные иконки — `#a18f7f`
- hover у кнопок в шапке — `#2b241f`
- брендовый квадрат / акцентный элемент — `#b48d72`

### 3. Карточки мероприятий
- фон карточки — `#221d19`
- border — `#42372f`
- hover фон — `#2b241f`
- заголовок карточки — `#f1e6d9`
- дата / метаинформация — `#a18f7f`
- hover-акцент заголовка — `#d8b69a`

### 4. Кнопки
#### Primary button
- фон — `#b48d72`
- текст — `#171411` или `#ffffff`
- hover — чуть темнее `#b48d72`

#### Secondary button
- фон — `#2b241f`
- border — `#42372f`
- текст — `#cdbbaa`
- hover — `#342b24`

### 5. Tabs / segmented controls
- неактивный фон — `#221d19`
- неактивный текст — `#cdbbaa`
- border — `#42372f`
- активный фон — `#b48d72`
- активный текст — `#171411` или `#ffffff`

### 6. Search / inputs
- фон input — `#221d19`
- border — `#42372f`
- основной текст — `#f1e6d9`
- placeholder — `#a18f7f`
- иконка поиска — `#a18f7f`
- focus border — `#b48d72`

### 7. Dropdown / menu / popup
- фон — `#221d19`
- border — `#42372f`
- текст — `#cdbbaa`
- hover item — `#2b241f`
- активный item / выделение — `#d8b69a`

### 8. Уведомления
- фон панели — `#221d19`
- border — `#42372f`
- основной текст — `#f1e6d9`
- вторичный текст — `#cdbbaa`
- время / слабый текст — `#a18f7f`
- акцент внутри уведомления — `#d8b69a`

### 9. Profile menu
- фон — `#221d19`
- border / separator — `#42372f`
- обычный текст — `#cdbbaa`
- hover — `#2b241f`
- logout / destructive — `#c76868`

### 10. Статусы
#### Draft
- текст — `#d8b69a`
- фон badge — полупрозрачный `#b48d72`

#### Ready / active
- текст — `#86906a`
- фон badge — полупрозрачный `#86906a`

#### Done / completed
- текст — `#9f8474`
- фон badge — полупрозрачный `#9f8474`

### 11. Empty states
- фон под иконку — `#2b241f`
- иконка — `#a18f7f`
- заголовок — `#f1e6d9`
- описание — `#a18f7f`
- CTA — `#b48d72`
