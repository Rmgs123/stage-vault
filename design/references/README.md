# Референсы дизайна

Примеры кода UI-компонентов, определяющие визуальный стиль проекта.
Агенты (Claude Code, Codex) используют эти файлы как образец при создании новых компонентов.

## Файлы

| Файл | Описание |
|------|----------|
| `design_code_example_1.jsx` | Экран входа / Регистрации / Сброс пароля|
| `design_code_example_2.jsx` | Дешборд проектов (мероприятий) |
| `design_code_example_3.jsx` | Профиль пользователя |
| `design_code_example_4.jsx` | Файлы Проекта |
| `design_code_example_5.jsx` | Таймлайн Проекта |
| `design_code_example_6.jsx` | Команда и Настройки Проекта |
| `design_code_example_7.jsx` | Пульт ведущего |
| `design_code_example_8.jsx` | Страница заглушка |

## Что здесь важно для агентов

- Цветовая палитра (HEX-коды из Tailwind-классов)
- Скругления (`rounded-xl`, `rounded-2xl`)
- Отступы и размеры (`px-4 py-3`, `gap-5`)
- Шрифты и размеры текста (`text-[15px]`, `font-semibold`)
- Стиль инпутов, кнопок, карточек
- Паттерны hover/focus/transition
- И другое...
