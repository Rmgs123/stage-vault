# Дизайн StageVault

## Экраны (`screens/`)

Экспорт в PNG. Именование по номерам из SPEC.md раздел 6.1:

| Файл | Экран |
|------|-------|
| `01-login.png` | Вход |
| `02-register.png` | Регистрация |
| `03-forgot-password.png` | Сброс пароля |
| `04-dashboard.png` | Список проектов |
| `05-event-files.png` | Файлы проекта |
| `06-event-timeline.png` | Сценарий (Timeline) |
| `07-event-team.png` | Команда |
| `08-event-settings.png` | Настройки проекта |
| `09-presenter.png` | Пульт ведущего |
| `10-code-entry.png` | Ввод кода (/go) |
| `11-profile.png` | Профиль |
| `12-mobile-stub.png` | Заглушка мобильных |

- Названия файлов скриншотов экранов могут отличаться, но все равно будут передавать суть своего содержимого в названии.

## Компоненты (`components/`)

Отдельные UI-элементы для точечной верстки:

```
Примеры:

audio-player.png
file-card.png
timeline-block.png
invite-dialog.png
ai-chat.png
...
```

- Не факт что содержимое папки /design будет совпадать с перечисленным в этом документе, ориентироваться по ситуации!
